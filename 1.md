### 题目1 （6分）

完善以下分段函数的程序，输入任意一个整数n，如果n>0 则输出其平方根（保留2位小数），如果n小于0，则输出其绝对值，否则输出0。（用你的代码替换题目中的下划线，不能使用math库）


```python
# 题目1代码，请不要改动已给出的代码

n = int((input('请输入一个整数：')))

if n > 0:

    print('{} 的平方根为：{:.2f}'.format(n, n**0.5))

elif n < 0:

    print('{}的绝对值为： {:.2f}'.format(n, -n))

else:

    print(0)
```
